数据大屏部署方案

# 数据大屏部署方案

大屏通常装有安卓系统，可以通过安装的app来打开应用。有以下几种部署方案：

# 一、浏览器 + H5

采用大屏安装的飞视浏览器直接访问

优点：浏览器安装好之后方便页面随时打开、切换

缺点：卡顿、渲染效果差

# 二、HBuilder打包安卓APP

使用HBuilder打包安卓app，安装到大屏系统中直接打开使用。

优点：浏览器内核提升，渲染效果增强

缺点：1.打包为在线打包，效率较低且代码安全性差 2.迭代及调试（反复安装）不方便

# 三、安卓APP + H5

安卓app加webview展示H5页面，app加入地址输入、切换功能。

优点：从渲染效果、版本迭代、在线调试等方面效果均不错。

缺点：小米盒子和小米电视系统差异，小米盒子安装的app不支持IP地址访问webview页面，只能再小米电视上进行在线调试。
